# Desafio Técnico

Projeto composto por 3 desafios:

1. Calculador de Salário
2. Sequencia de Numeros
3. Sequencia de Fibonacci

# Tecnologias

- JavaScript
- React
- TypeScript
- Styled-components
- Vite

Todos os desafios haviam requisitos de valores e impressões de resultados. Foram realizados tudo dentro dos padrões solicitados e abaixo segue as instruções para testes.

# Clonagem do projeto para rodar localmente

1- Faça a clonagem do repositório
- git clone https://github.com/leandrobss1/Teste-Tecnico-AceData.git

2- Acesse a pasta do projeto
- cd Teste-Tecnico-AceData

3- Instale as dependências
- npm install

4- Execute o projeto
- npm run dev
